## 🎯 Overview

Welcome to the **Agentic Premier League 2026**! This is a cutting-edge AI buildathon where participants develop intelligent agents that compete in simulated cricket matches. Build your agentic system, optimize strategies, and lead your team to victory!

### What is This?
- **Competition Type**: AI Agent Development & Competition
- **Format**: Teams compete under RCB or MI banners
- **Goal**: Build the most effective agentic system for cricket simulation
- **Duration**: Check SUBMISSION_GUIDELINES.md for deadlines

## 🚀 Getting Started

1. **Choose Your Team**: RCB or MI
2. **Read the Guidelines**: Check SUBMISSION_GUIDELINES.md
3. **Review Tracks**: See TRACKS.md for difficulty levels
4. **Use the Template**: Copy PROJECT_TEMPLATE.md to start
5. **Submit Your Project**: Follow the submission checklist

Each team folder contains:
- Source code
- README
- Architecture
- Demo links
- Screenshots
- Deployment links
---

# Rules

- Teams must attempt all 5 mandatory features
- Partial submissions are allowed
- Any tech stack/framework/tools are allowed
- Focus on scalability, usability, automation, and innovation

# Problem Statements

## 1. AI Career Copilot for Students

Build an AI-powered Career Copilot that helps students with:
- Resume analysis
- Internship matching
- Mock interviews
- Skill recommendations
- Career growth tracking

## 2. Autonomous Hiring & Talent Intelligence Platform

Build an AI hiring platform that automates:
- Candidate screening
- Talent ranking
- Interview workflows
- AI communication
- Recruitment analytics
