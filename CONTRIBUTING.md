# Contributing Guidelines

Welcome to Agentic Premier League 2026 🚀

## Branch Naming Convention

Each team should create their own branch:

```txt
team-project
```

Example:
```txt
team-Diva-copilot
```

---

# Folder Rules

RCB teams must add projects inside:

```txt
/rcb/
```

MI teams must add projects inside:

```txt
/mi/
```

---

# Commit Naming Convention

Use meaningful commit messages:

```txt
feat: added resume analyzer
fix: resolved auth issue
ui: improved dashboard
docs: updated architecture
ai: integrated Gemini agent
```

---

# Pull Request Workflow

1. Fork repository
2. Clone repository
3. Create branch
4. Add project folder
5. Push changes
6. Create PR

---

# Mandatory Files

Every team MUST include:

- README.md
- Architecture diagram
- Setup instructions
- Demo link
- Team details

---

# Daily Contribution Requirement

Teams are encouraged to:
- Push regular commits
- Track progress daily
- Maintain clean commit history

---

# Do Not

❌ Push directly to main  
❌ Upload broken builds  
❌ Create random folder structures  
❌ Skip README documentation
