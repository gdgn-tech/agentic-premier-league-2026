# Submission Guidelines

## Mandatory Submission Items

Every team must submit:

- Source code
- README
- Architecture diagram
- Demo video
- Deployment link
- Presentation/PPT

---

# Submission Folder Format

```txt
rcb/team-project/
```

OR

```txt
mi/team-project/
```

---

# Judging Focus

Projects will be evaluated on:

- Innovation
- Scalability
- Agentic Architecture
- Product Thinking
- User Experience
- Technical Execution
