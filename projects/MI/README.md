# MI Projects

This directory contains all projects submitted by teams competing under the **Mumbai Indians** banner.

## Structure

```
MI/
├── team-name-1/
├── team-name-2/
└── ...
```

Teams should create their project directories following the naming convention: `your-team-name/`

Refer to [PROJECT_TEMPLATE.md](../PROJECT_TEMPLATE.md) for the required project structure.
